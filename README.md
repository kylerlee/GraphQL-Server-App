# GraphQL-Server-App

A practice of server application development using GraphQL, MongoDB, NestJS and TypeORM.

## Getting Started

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev
```

## Test

```bash
# unit tests
$ npm run test
```
